Importação de pedidos (encomendas) usando a planilha modelo

Utilize o modelo padronizado pela Mandaê para cadastrar suas encomendas fazendo o envio de uma planilha.

Última atualização: junho de 2025

A importação por planilha é a forma mais prática de cadastrar múltiplas encomendas de uma só vez. Basta preencher o modelo padronizado com os dados dos seus destinatários e fazer o upload pelo menu de Coletas, sem precisar inserir pedido por pedido manualmente.

Download do modelo de planilha

Entre no menu de encomendas e selecione a opção "Planilha". No box cinza que irá aparecer, clique no link chamado "planilha padronizada".

1
Acesse o menu de Coletas

Clique no ícone de caminhão no menu lateral.

2
Selecione "Planilha"

Escolha a opção "Planilha" entre os métodos disponíveis.

3
Baixe a planilha padronizada

No box "Primeira vez?", clique em "planilha padronizada".

Observações sobre o preenchimento

  • O seu arquivo deverá conter colunas idênticas à planilha padronizada da Mandaê. Caso contrário, uma mensagem de erro indicará quais campos estão incorretos e como corrigir.
  • Cada linha corresponde a uma encomenda diferente. Pedidos com mais de uma encomenda para o mesmo destinatário são considerados encomendas diferentes.
  • Você poderá incluir até 500 itens no mesmo arquivo. Para solicitações de coleta com mais de 500 itens, basta importar uma nova planilha com os pedidos restantes clicando em "Adicionar nova encomenda" após fazer o upload da primeira planilha.
⚠
O campo QR Code não deve ser utilizado. Deixe-o vazio para evitar erros no processamento das encomendas.

Envio da planilha e verificação de problemas

A planilha preenchida deve ser enviada através da mesma página onde o modelo foi baixado, pelo menu "Coletas".

  1. Escolha o ícone "Planilha" no menu de encomendas.
  2. Na página seguinte, clique no botão "Adicione o arquivo" e selecione a planilha do seu computador.
  3. Aguarde o carregamento completo do arquivo.
ℹ
Caso haja problemas de preenchimento em alguma linha (encomenda), você terá que corrigir e enviar novamente a planilha completa, não apenas as linhas com erro.

Tipo de arquivo

Aceitamos os seguintes formatos de arquivo:

  • .XLS e .XLSX: arquivos do Microsoft Excel.
  • .CSV: provenientes da exportação de pedidos via sistemas de ERP ou plataformas de e-commerce.
⚠
A importação via .CSV precisa conter os mesmos campos da planilha padronizada da Mandaê e usar ponto e vírgula ( ; ) como separador entre as colunas. Arquivos com separador vírgula ou tabulação não serão reconhecidos corretamente.

Verificação de erros

Após o upload, o sistema valida cada linha da planilha automaticamente. Se houver inconsistências, você verá indicações visuais de quais encomendas precisam de correção.

  • Corrija os campos indicados diretamente na planilha original.
  • Salve o arquivo e faça um novo upload completo, com todas as encomendas, inclusive as que não tinham erro.
  • Repita o processo até que todas as encomendas sejam validadas com sucesso.
✓
Quando todas as linhas forem validadas sem erros, suas encomendas serão criadas automaticamente e estarão prontas para coleta.
